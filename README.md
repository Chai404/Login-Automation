Login Automation using Selenium:-

This project automates the login process on a sample practice website using:
- Python
- Selenium WebDriver
- XPath, ID locators
- Explicit waits

Features:
1. Opens Chrome browser  
2. Enters username and password  
3. Clicks login button  
4. Validates success message  

How to run:
1. Install Python
2. pip install selenium
3. Update ChromeDriver
4. Run: python login.py
